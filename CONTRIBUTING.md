# Contributing

When reporting a bug you _must_ provide a standalone script of sort that
reproduces the problem you're encountering. Bug reports along the lines of "X
doesn't work" without any additional details will be closed.

This plugin is based around my own needs, it just happens to be a standalone
plugin so others may also use it. I consider this plugin to be feature complete
and as such won't accept new features _unless_ I happen to have a need for them
myself.
